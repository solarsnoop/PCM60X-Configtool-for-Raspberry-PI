# PCM60X-Configtool-for-Raspberry-PI
Python 3 Code for detect automatic avaible PCM60X devices and change easy Bulk, Float and current

1. please install on your raspberry Pi: 
pip install pyserial questionary --break-system-packages

after you can run this program with: python3 pcm60x_config.py

Attention: 
First answer how many cells you are using with your accu: Exemplam S8 = 8, S16=16, ... Its a small help for that you can see how many Voltage each cell you have with your setup
